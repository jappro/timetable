# 🔒 Lock In — 180 Day Daily Timetable

A personal productivity dashboard built as a single-file HTML page. Designed to be opened every day as a reference and accountability tool during a focused 180-day execution period.

-----

## What This Is

This is not a fancy app. It is a **daily operating system** — a visual timetable that keeps your schedule, build rotation, study plan, and mindset locked in one place. Every time you open it, the session timer starts counting, reminding you that time is already moving.

-----

## Features

- **Full 24-hour daily schedule** — every block labeled, color-coded, and tagged
- **Scramble-then-lock session timer** — counts hours, minutes, seconds, and milliseconds from the moment you open the page; starts in chaotic scramble mode and locks into real time
- **Build rotation tracker** — cycles through Vinceslink.com → Alfalf AI → Portfolio, with priority override logic
- **Skill study rotation** — Data Analysis (Mon/Wed/Fri), Marketing (Tue/Thu), AI Automation (Sat/Sun)
- **Weekly study calendar** — full 7-day view at a glance
- **Zero dependencies** — one `.html` file, loads from Google Fonts, works offline after first load
